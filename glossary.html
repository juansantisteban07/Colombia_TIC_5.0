32 términos técnicos aprendidos en Colombia TIC 5.0

ES/EN Bilingüe
# Término en Inglés Traducción al Español Definición Categoría
01 Artificial Intelligence Inteligencia Artificial Simulación de procesos de inteligencia humana por sistemas informáticos, incluyendo aprendizaje, razonamiento y autocorrección. Simulation of human intelligence processes by computer systems, including learning, reasoning and self-correction. IA
02 Machine Learning Aprendizaje Automático Rama de la IA que permite a los sistemas aprender y mejorar a partir de la experiencia sin ser programados explícitamente. Branch of AI that enables systems to learn and improve from experience without being explicitly programmed. IA
03 Generative AI IA Generativa Sistemas de IA capaces de generar nuevo contenido como texto, imágenes, audio o video basándose en datos de entrenamiento. AI systems capable of generating new content such as text, images, audio or video based on training data. IA
04 Neural Network Red Neuronal Sistema computacional inspirado en el cerebro humano, compuesto por nodos interconectados que procesan información. Computing system inspired by the human brain, composed of interconnected nodes that process information. IA
05 Large Language Model Modelo de Lenguaje Grande Modelo de IA entrenado en enormes cantidades de texto para entender y generar lenguaje humano de manera natural. AI model trained on vast amounts of text data to understand and generate human language naturally. IA
06 Cybersecurity Ciberseguridad Práctica de proteger sistemas, redes y programas de ataques digitales, daños o acceso no autorizado. Practice of protecting systems, networks and programs from digital attacks, damage or unauthorized access. Seguridad
07 Zero Trust Confianza Cero Marco de seguridad que requiere verificación estricta de identidad para cada persona y dispositivo que intente acceder a recursos. Security framework requiring strict identity verification for every person and device trying to access resources. Seguridad
08 Phishing Suplantación de identidad Intento fraudulento de obtener información sensible suplantando a una entidad de confianza mediante correos o sitios falsos. Fraudulent attempt to obtain sensitive information by disguising as a trustworthy entity through fake emails or sites. Seguridad
09 Ransomware Software de Secuestro Software malicioso que cifra los archivos de la víctima y exige un pago para recuperar el acceso a los datos. Malicious software that encrypts victim's files and demands payment for the decryption key to recover access. Seguridad
10 Encryption Cifrado Proceso de codificar información para que solo las partes autorizadas puedan acceder a ella, protegiéndola de terceros. Process of encoding information so only authorized parties can access it, protecting it from unauthorized third parties. Seguridad
11 Cloud Computing Computación en la Nube Entrega de servicios de cómputo a través de internet, incluyendo almacenamiento, procesamiento y software. Delivery of computing services over the internet, including storage, processing and software on demand. Cloud
12 Serverless Sin Servidor Modelo de ejecución en la nube donde el proveedor gestiona la infraestructura del servidor automáticamente. Cloud execution model where the cloud provider manages server infrastructure automatically without manual configuration. Cloud
13 SaaS Software como Servicio Modelo de distribución de software donde las aplicaciones se alojan en la nube y se ofrecen a los usuarios por internet. Software distribution model where applications are hosted in the cloud and provided to users over the internet. Cloud
14 Infrastructure as Code Infraestructura como Código Gestión y aprovisionamiento de infraestructura computacional a través de archivos de configuración legibles por máquinas. Managing and provisioning computing infrastructure through machine-readable configuration files instead of manual processes. Cloud
15 Frontend Interfaz de Usuario La parte de un sitio web o aplicación con la que los usuarios interactúan directamente, construida con HTML, CSS y JavaScript. The part of a website or application that users interact with directly, built with HTML, CSS and JavaScript. Desarrollo
16 Backend Lógica del Servidor El lado del servidor de una aplicación web que maneja la lógica de negocio, bases de datos y gestión del servidor. The server-side of a web application, handling business logic, databases and server management behind the scenes. Desarrollo
17 API Interfaz de Programación Conjunto de reglas y protocolos que permite que diferentes aplicaciones de software se comuniquen entre sí. Set of rules and protocols that allows different software applications to communicate and share data with each other. Desarrollo
18 Responsive Design Diseño Responsivo Enfoque de diseño web que hace que las páginas se vean bien en diferentes tamaños de pantalla y dispositivos. Web design approach that makes web pages render well on different screen sizes and devices automatically. Desarrollo
19 Version Control Control de Versiones Sistema que registra los cambios en archivos a lo largo del tiempo para poder recuperar versiones anteriores cuando se necesite. System that records changes to files over time so you can recall specific versions later when needed. Desarrollo
20 Big Data Grandes Datos Conjuntos de datos extremadamente grandes que requieren herramientas y técnicas especiales para analizarlos y extraer valor. Extremely large datasets that require special tools and techniques to analyze and extract meaningful value from them. Datos
21 Data Mining Minería de Datos Proceso de descubrir patrones y conocimientos útiles en grandes cantidades de datos mediante algoritmos estadísticos. Process of discovering patterns and useful knowledge from large amounts of data using statistical algorithms. Datos
22 Data Lake Lago de Datos Repositorio centralizado que almacena grandes cantidades de datos en su formato nativo hasta que sean necesarios para su análisis. Centralized repository that holds vast amounts of raw data in its native format until needed for analysis. Datos
23 Data Privacy Privacidad de Datos El derecho de las personas a controlar cómo se recopila, usa y comparte su información personal por parte de organizaciones. The right of individuals to control how their personal information is collected, used and shared by organizations. Datos
24 Internet of Things Internet de las Cosas Red de dispositivos físicos conectados a internet que recopilan y comparten datos entre sí de forma automática. Network of physical devices connected to the internet, automatically collecting and sharing data with each other. Redes
25 Bandwidth Ancho de Banda Velocidad máxima de transferencia de datos a través de una conexión de red en un período de tiempo determinado. Maximum rate of data transfer across a network connection during a specific period of time. Redes
26 Firewall Cortafuegos Dispositivo de seguridad de red que monitorea y controla el tráfico entrante y saliente según reglas de seguridad predefinidas. Network security device that monitors and controls incoming and outgoing network traffic according to predefined security rules. Redes
27 Digital Transformation Transformación Digital Integración de tecnología digital en todas las áreas de un negocio, cambiando fundamentalmente cómo opera y entrega valor. Integration of digital technology into all areas of a business, fundamentally changing how it operates and delivers value. Gestión
28 Agile Methodology Metodología Ágil Enfoque iterativo para la gestión de proyectos y el desarrollo de software que ayuda a los equipos a entregar valor más rápido. Iterative approach to project management and software development that helps teams deliver value faster and adapt to change. Gestión
29 Interoperability Interoperabilidad Capacidad de diferentes sistemas y organizaciones para trabajar juntos e intercambiar información de manera efectiva. Ability of different systems and organizations to work together and exchange information effectively and seamlessly. Gestión
30 Scalability Escalabilidad Capacidad de un sistema para manejar mayor carga de trabajo agregando recursos de manera proporcional a la demanda. Ability of a system to handle increased workload by adding resources proportionally to meet growing demand. Gestión
31 Motion Capture Captura de Movimiento Tecnología que registra los movimientos de objetos o personas y los traduce en modelos digitales tridimensionales. Technology that records the movements of objects or people and translates them into three-dimensional digital models. Animación
32 Rendering Renderizado Proceso de generar una imagen fotorrealista o no fotorrealista a partir de un modelo 2D o 3D utilizando software de computadora. Process of generating a photorealistic or non-photorealistic image from a 2D or 3D model using computer software. Animación
